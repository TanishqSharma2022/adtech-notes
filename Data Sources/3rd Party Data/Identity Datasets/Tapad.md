
This dataset contains identity graphs of users. It includes household ids and what all individuals are in that household and what devices they have.

Types are
- APAC - Asia Pacific
- NA - North America

Includes following columns: 
- household  - links all individuals and devices within the same household.
- individual id - identifies each person in a household.
- ip
- country code
- device id
- Household quality score
- individual device quality score
- Primary Hardware - Mobile Phones, Desktop, etc
- Device ID
