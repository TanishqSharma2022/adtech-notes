
The main components of agents are 
1. Thinking: LLM part of the agent thinks what the next step should be
2. Action : LLM use tools to act 
3. Observation: Observe the output and prepares an answer for the user. 

Common Thought Types:

Chain-of-Thought


**ReAct Approach**
This approach means Reasoning(Thinking) with Acting.
React is a prompting technique that encourages the agent to think step-by-step and use tools for actions.
