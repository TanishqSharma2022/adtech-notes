
- **Workflows** are systems where LLMs and tools are orchestrated through predefined code paths.
- **Agents**, on the other hand, are systems where LLMs dynamically direct their own processes and tool usage, maintaining control over how they accomplish tasks.

## LlamaIndex

It is a full toolkit to make agentic systems with your data through indexes and workflows. 
There are three parts that we should be focusing on: 
- Components
- Agents 
- Tools and Workflows


**Components**: Basic Building blocks in the agentic systems that includes prompts, models and databases. 

**Tools**: Help the agent to perform tasks such as searching, calculating etc. 

**Agents**: Autonomous components that use tools to perform actions and make decisions for complex problems. 

**Workflows**: Step-by-step processes that process logic altogether. 

**LlamaIndex** has hundreds of these components, agents and tools predefined in **LlamaHub**
